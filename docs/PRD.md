# 부경대 학식 알림 서비스 PRD/TRD

## PRD (Product Requirements Document)

### 1. 개요
부경대학교 학식 메뉴 자동 수집 및 안드로이드 앱 서비스

### 2. 문제 정의
- 매번 학교 홈페이지 들어가서 식단 확인하는 게 번거로움
- 새 식단이 올라왔는지 수동으로 체크해야 함

### 3. 목표
- 새로운 식단 게시물 자동 감지 및 수집
- 식단 데이터를 DB에 저장하여 앱에서 조회 가능하게

### 4. 사용자
- 부경대 학생/교직원

### 5. 핵심 기능

**백엔드 (크롤러)**
1. 자동 크롤링: 정해진 시간에 식단 게시판 체크
2. 중복 방지: 이미 수집한 게시물 스킵
3. 텍스트 파싱: 라일락 식단표 텍스트 추출
4. DB 저장: Supabase에 메뉴 데이터 + 로그 저장

**프론트엔드 (안드로이드 앱)**
1. 라일락 식단 조회
2. 좌우 스와이프로 요일별 메뉴 확인
3. 하단 광고 배너

### 6. 성공 지표
- 새 게시물 올라오면 1시간 내 DB 반영
- 앱에서 최신 식단 정상 조회
